Location

This board tracks trains, buses, and planes near one location — Bethesda, MD 20816 by default. Type any address, city, or zip to move it there instead. Metro/MARC/Ride On only cover the DC–Maryland area, but Amtrak and planes work anywhere in the US.

One-time setup — WMATA API key

Train & bus times come from WMATA (Washington Metro). It's free: create an account, then subscribe to the “Default Tier” product and copy your Primary key.

→ developer.wmata.com/signup, then subscribe to Default Tier, then copy your key here.
